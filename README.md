
#npoco

npoco is a node.js command tools to generate petapoco(micro-ORM) classes for .NET and Mono



See here - https://github.com/toptensoftware/PetaPoco - for PetaPoco details.


##usage

1.install  npoco
	
	npm install -g npoco

1.init a config 

	npoco init


2.gen PetaPoco's code

	npoco gen
	

###help



   - npoco -help
   - npoco init -h
   - npoco gen -h
